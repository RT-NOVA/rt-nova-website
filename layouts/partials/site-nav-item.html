{{- $item := .item -}} {{- $level := default 0 .level -}} {{- $id := .id -}} {{- $children := default (slice) $item.children -}} {{- $hasChildren := gt (len $children) 0 -}} {{- if and $hasChildren (eq $level 0) -}}
{{- with $item.url }} {{ $item.name }} Overview {{- end }} {{- range $childIndex, $child := $children }} {{- partial "site-nav-item.html" (dict "item" $child "level" 1 "id" (printf "%s-%d" $id $childIndex)) -}} {{- end }}
{{- else if $hasChildren -}}
{{- with $item.url }} {{ $item.name }} {{- else }} {{ $item.name }} {{- end }}
{{- range $childIndex, $child := $children }} {{- partial "site-nav-item.html" (dict "item" $child "level" (add $level 1) "id" (printf "%s-%d" $id $childIndex)) -}} {{- end }}
{{- else if eq $level 0 -}} {{ $item.name }} {{- else -}} {{ $item.name }} {{- end -}}
