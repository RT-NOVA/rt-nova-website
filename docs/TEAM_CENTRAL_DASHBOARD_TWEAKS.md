# Team Central dashboard tweaks

This update changes Team Central from a season dropdown to a two-option Current / Upcoming selector, removes the extra explanatory text, simplifies the dark season summary area, fixes dark badges on dark backgrounds, and adds banded table rows for readability.

Apply from repo root:

```bash
unzip -o rt-nova-team-central-dashboard-tweaks.zip -d .
python3 scripts/apply-team-central-dashboard-tweaks.py
hugo server -D
```
