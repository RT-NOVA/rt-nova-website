# Coaching Opportunities Page

The `/coaching-opportunities/` page is powered by:

- `content/coaching-opportunities.md`
- `data/coaching_opportunities.yaml`
- `layouts/partials/page-coaching-opportunities.html`
- `assets/css/coaching-opportunities.css`
- `static/downloads/coaching/head-coach-listing.pdf`
- `static/downloads/coaching/assistant-coach-listing.pdf`

## Updating role content

Update text, role highlights, comparison rows, standards, support items, and application steps in `data/coaching_opportunities.yaml`.

## Updating PDFs

Replace the files in `static/downloads/coaching/` while keeping the same names:

- `head-coach-listing.pdf`
- `assistant-coach-listing.pdf`

## Color notes

This page uses the current black / charcoal / orange theme. It does not add blue-toned color values or reset global CSS variables.
